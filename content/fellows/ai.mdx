### Example: Understanding Attention

Attention mechanisms allow models to focus on relevant parts of input sequences:

```python
# Simplified attention mechanism
def attention(query, key, value):
    scores = query @ key.T / sqrt(d_k)
    weights = softmax(scores)
    output = weights @ value
    return output
```

AI Fellow explains not just *what* attention is, but *why* it works and *when* to use it.
