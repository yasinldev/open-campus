# Open Campus Images

This directory contains images for the Open Campus platform.

## Directory Structure

```
public/
└── images/
    ├── fellows/        # Fellow avatars
    ├── blog/          # Blog post cover images
    └── og-image.png   # Default Open Graph image
```

## Image Guidelines

### Fellows
- Size: 400x400px
- Format: PNG with transparent background
- Style: Consistent with design system

### Blog
- Size: 1200x630px (Open Graph standard)
- Format: JPG or PNG
- Content: Relevant to article topic

### General
- Optimize all images before committing
- Use descriptive filenames
- Include alt text in components

## Placeholder Images

For development, use placeholder services:
- https://via.placeholder.com/400x400
- https://images.unsplash.com/ (with proper attribution)
